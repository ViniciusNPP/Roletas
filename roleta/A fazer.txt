ROLETA
    1- MUDAR O BACKGROUND DOS LABELS DINAMICAMENTE ATRAVÉS DO COLOR-PREVIEW (PARA FICAR MAIS FÁCIL DE VISUALIZAR AS CORES)
    2- FUNÇÕES DE CONTROLAR O TAMANHO DOS LABELS (TANTO O RAIO QUANTO O TAMANHO DA FONTE) QUANDO O NOME FOR AUMENTANDO DE TAMANHO
    3- CRIAR FUNÇÃO PARA CONTROLAR AS CONFIGURAÇÕES DA ROLETA
    4- FAZER COM QUE OS CONTAINERS ITEMS E CONFIG SÓ APAREÇAM QUANDO CLICAR NO BOTÃO DE ITEMS E CONFIGURAÇÕES, RESPECTIVAMENTE
    4.1- OS CONTAINERS DEVEM SE SOBRESSAIR PERANTE A ROLETA
    4.2- O FUNDO DEVE FICAR MAIS ESCURO QUANDO OS CONTAINERS ESTIVEREM ABERTOS
    4.3- OS CONTAINERS NÃO DEVEM FECHAR QUANDO CLICAR FORA DELES, SOMENTE NO BOTÃO DELES OU, SE TIVER, EM UM BOTÃO DE PRONTO/SALVAR/ETC 
    5- FUNÇÃO PARA SALVAR OS DADOS DA ROLETA NO LOCALSTORAGE E FUNÇÃO PARA CARREGAR OS DADOS DA ROLETA DO LOCALSTORAGE
    6- DEIXAR BONITINHO

    COR_PREVIEW
        1- CRIAR DIV PARA MOSTRAR NO CENTRO DA TELA
        2- CRIAR MENU PARA SELECINAR A COR
        3- VERIFICADOR DE LUMINOSIDADE
        4- ALTERAR A COR DO TEXTO DE ACORDO COM A LUMINOSIDADE
        5- ALTERAR O QUADRADO DO COR-PREVIEW COM A COR ESCOLHIDA NO IRO
        6- APLICAR A COR ESCOLHIDA AO LABEL
    
    CONFIGURAÇÕES
        1- CRIAR MENU COM TODAS OS NOMES DAS CONFIGURAÇÕES
        2- TORNAR POSSÍVEL A EDIÇÃO DAS CONFIGURAÇÕES DA ROLETA
        3- CRIAR CONFIGURAÇÃO QUE PERMITE O USUÁRIO CRIAR UMA PALETA DE CORES
            3.1- INSERÇÃO DA COR INICIAL E DA COR FINAL
            3.2- A QUANTIDADE DE CORES NESSA PALHETA SERÁ A QUANTIDADE DE PROPS DA ROLETA

DENTRO DA PASTA
    1- ADICIONAR MAIS OPÇÕES DE AÇÕES AO CONTEXT MENU DOS ITENS
    2- ADICIONAR BOTÃO DE DUPLICAR ITEM

GERAL
    1- LINKAR COM A PASTA ESPECÍFICA DE CADA ROLETA
    2- PERMITER QUE O USUÁRIO BAIXE UM ARQUIVO JSON PARA CASO O LOCALSTORAGE FIQUE CHEIO OU QUEIRA ACESSAR DE OUTRO LOCAL
